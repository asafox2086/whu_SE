# 星轨平台 MVP

星轨平台 MVP 用于验证学生发现竞赛/科研机会、发布组队招募、申请科研项目和维护证书记录的核心流程。

## 项目结构

- `public/`: 静态页面和样式
- `src/`: MVP 领域逻辑和浏览器交互
- `tests/`: 核心流程行为测试
- `scripts/`: 本地启动脚本
- `MVP/`: 课程要求的优先级清单和验证反馈报告
- `CONTEXT.md`: 项目领域词汇表

## 环境要求

- Node.js 20 或更高版本
- npm 10 或更高版本

## 运行

在仓库根目录执行：

```bash
npm install
npm start
```

启动后访问：

```text
http://localhost:4173
```

如需换端口：

```bash
PORT=4300 npm start
```

Windows PowerShell:

```powershell
$env:PORT=4300; npm start
```

## 测试

在仓库根目录执行：

```bash
npm test
```

测试覆盖：

- 学生注册与登录
- 竞赛/科研机会浏览与筛选
- 组队招募发布与竞赛详情展示
- 科研项目申请与导师侧查看
- 证书记录上传校验与列表展示

## MVP 范围

已完成：

- 学生本地注册登录
- 注册时选择权限身份：学生、导师、证书收集者
- 机会大厅
- 竞赛组队招募
- 科研申请与导师侧演示队列
- 学生证书记录保存，证书收集者汇总查看
- 使用信息记录与手动反馈收集

暂不包含：

- 真实后端数据库
- 邮箱验证
- 真实文件上传到对象存储
- 多角色权限后台
- 身份真实性核验

## 数据说明

当前 MVP 使用浏览器 `localStorage` 保存演示数据。需要清空数据时，可在浏览器开发者工具中删除 `xinggui_mvp_state_v1` 和 `xinggui_mvp_session_v1`。
